## 🚀 Featured Projects

### 🛠️ Raspberry Pi Pico DMX Controller
A custom hardware solution built to solve work light issues in our theater's control booth. It outputs DMX512 signals to control LED PAR work lights via programmed presets (Full White / Crew Blue / Off).
* **Tech Used:** Raspberry Pi Pico, RS-485 translator board, MicroPython
* 📂 **[View Repository]([Link to your Pico Repo](https://github.com/estech321/pico-dmx-panel))**

### 🔍 Keypad and RFID Access Control System
An experimental device based on a Raspberry Pi Pico that requires both an RFID ID card and a PIN to be entered before allowing access. Key codes can be used to modify allowed tags and PINs. Use this to secure whatever you want! (Just note that it is not *actually* secure due to many limitations of the Pico and code.)
* **Tech Used:** Raspberry Pi Pico 2, RFID reader, keypad, MicroPython
* 📂 **[View Repository]([Link to Repo 2](https://github.com/estech321/keypad-access-control))**
